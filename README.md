### GY-H5App
   这是贵阳的一个移动端和pc端的项目
   Ocps-Project--移动端
   Ocps-Project-pc--pc端
### 里面包含了：
   1、生成二维码
   
   2、扫码
   
   3、弹框
   
   4、加密
   
   5、日历插件
   
   6、正则验证
   
   7、键盘
### 运行命令：
 1、num install ---安装插件
 
 2、其余的安装请查看插件安装文档
 
 3、npm run dev
 
 4、打包 npm run build
